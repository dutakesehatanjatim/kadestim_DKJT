## ✨ Fitur Unggulan

### 👥 Untuk Pengunjung Umum
- ✅ Melihat daftar event kesehatan terbaru
- ✅ Membaca informasi dan artikel kesehatan
- ✅ Desain modern & responsif (mobile friendly)
- ✅ Navigasi yang mudah dan intuitif

### 🔐 Untuk Admin
- ✅ Login sistem dengan autentikasi
- ✅ Menambah event kesehatan baru
- ✅ Mengedit event yang sudah ada
- ✅ Menghapus event yang tidak diperlukan
- ✅ Menambah informasi kesehatan
- ✅ Mengedit artikel kesehatan
- ✅ Menghapus informasi kesehatan
- ✅ Data tersimpan di localStorage (bisa diintegrasikan ke backend)
